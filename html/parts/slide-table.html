API для верстки простых таблиц

  • Исходная таблица сохраняется в формате .csv

    На входе — обычная таблица из Excel / Google Tables.

  • Полученный файл копируется в проект

    и может служить источником данных для построения таблиц или графиков.

  • Настройка параметров вывода (опция)

    Предусмотрена возможность выборки данных и разворота таблицы на 90°.

  • На странице автоматически строится таблица

    API предоставляет готовую верстку c данными из вашего файла.
